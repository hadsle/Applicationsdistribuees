Pour démarrer le projet, veuillez creez un base de donnees   dans notre client SQL pour créer le test de base de données avec ses données.

Tout d’abord, vous devez ajouter le fichier mysql-connector.jar au classpath

Puis exécutez jdbc.java en utilisant la commande javac jdbc.java
